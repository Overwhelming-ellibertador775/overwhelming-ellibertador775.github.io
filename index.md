## 🔧 Troubleshooting common issues

If the tool does not work, check these common items:

- **Connection errors:** Check your internet. Ensure your firewall allows the app to connect.
- **Authentication errors:** Check your API key. Copy and paste the key again to prevent typos.
- **API limits:** If you create many invoices, the FIZ server might pause your access. Wait five minutes and try again.
- **NIF errors:** Ensure the client NIF contains exactly nine digits.
